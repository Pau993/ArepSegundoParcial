# AREP Segundo Parcial

En este parcial desarrollamos una aplicación para calcular la **secuencia de Lucas**, la cual se define mediante la siguiente fórmula:

* L(0) = 2
* L(1) = 1
* L(n) = L(n−1) + L(n−2), para n ≥ 2
